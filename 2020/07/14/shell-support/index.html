cat

-E 显示行结束符$

-n 对显示出的每一行进行编号(包含空行)

-A 显示所有控制符(^I 代表table键)

-b 非空行编号

-s 压缩连续的空行成一行

more

(空格一页一页的翻,回车一行一行的翻) q 退出
less ：一页一页地查看文件

head

-c #：指定获取前#字节

-n#：指定获取前#行 (n 可以省略 -#)

tail

-c#：指定获取后#字节

-n#：指定获取后#行 (n 可以省略 -#)

-f：跟踪显示文件新追加的内容,常用日志监控

cut

-d 指名分隔符,默认tab
-f # ：第#个字段

#,#,#：离散的多个字段,例如1,3,7

#-#：连续的多个字段,例如1-4(也可以和离散字段一起使用例如：1-4,7)

-c ：按字符切割

wc

选项：-l 只计数行数(包含空行)

-w 只计数单词总数

-c 只计数字节总数

-m 只计数字符总数

sort

文本排序: 把整理过的文本显示在输出,不改变原始文件

常用选项 -r 执行反方向(由上至下)整理

-n 执行按数字大小整理

-f 选项忽略字符串中的字符大小写
-u 删除输出中的重复行(重复行全部不显示)
-t c 使用c作为字段界定符
-k X 按照使用c字符分隔的X列来整理能够使用多次。

uniq

uniq -c 显示每行重复出现的次数

-d 仅显示重复过的行

-u 仅显示不曾重复的行

常和sort命令一起配合使用,例： sort 文件名 |uniq -c

grep

–color=auto: 对匹配到的文本着色显示

-v: 显示不能够被pattern 匹配到的行(不包含匹配字的行)

-i: 忽略字符大小写

-n: 显示匹配的行号

-c: 统计匹配的行数

-o: 仅显示匹配到的字符串

-A #: 匹配行的后#行

-B #： 匹配行的前#行

-C #: 匹配行的前后各#行

-e ：实现多个选项间的逻辑 or 关系

-w ：整行匹配整个单词

-E ：使用扩展正则表达式

sed

-n 不输出模式空间内容到屏幕,即不自动打印(默认会把源文件打印)

-f 文件 ：从指定文件中读取编辑脚本
-r：支持使用扩展正则表达式
-i：原文编辑(会写入文件中)

awk是一种处理文本的编程语言工具，报告生成器，格式化文本输出

awk工作原理
第一步：执行BEGIN{action;处理内容}语句块中的语句（可省）
第二步：从文件或标准输入（stdin）读取一行，然后执行pattern{action;处理内容}语句块，它逐行扫描文件，从第一行到最后一行重复这个过程，直到文件全部被读取完毕
第三步：当读至输入流末尾时，执行END{action;处理内容}语句块
BEGIN语句块在awk开始从输入流中读取之前被执行，这是一个可选的语句块，比如变量初始化、打印输出表格的表头等语句通常写在BEGIN语句块中
END语句块在awk从输入流中读取完所有的行之后即可被执行，比如打印所有行的分析结果这类信息汇总都是在END语句块中完成，它也是一个可选语句块
pattern语句块中的通用命令是最重要的部分，也是可选的，如果没有提供pattern语句块，则默认执行{print}，即打印每一个读取到的行，awk读取的每一行都会执行该语句块
print格式：print item1，item2…（item条目）

要点：

  1. 逗号分隔符
  2. 输出的各item可以是字符串，也可以是数值；当前记录的字段、变量或awk的表达式
  3. 如省略item，相当于print $0
    实例
    无论输入什么都打印hello,awk 指定分隔符打印所有 指定分隔符打印指定列 指定分隔符打印第一列和第七列以table键分开 变量：内置和自定义变量
    FS：输入字符分隔符,默认为空白字符 OFS:输出字符分隔符,默认为空白字符 NF:字段数量 NR：行号 FILENAME：当前文件名 ARGC：命令行参数的个数 ARGV：数组，保存的是命令行所给定的各参数（0为第一个参数，1为第二个参数，以此类推）

awk格式化

printf命令
格式输出：printf "FORMAT",item1,item2…
必须指定FORMAT（格式）
不会自动换行，需要显示给出换行控制符 \n
FORMAT中需要分别为后面每个item指定格式符
格式符：与item一一对应
%d,%i：显示十进制整数
%s：显示字符串
%%：显示%自身

修饰符
#：打印#个字符宽度（可以为小数）
-：左对齐%-15s（默认不带 - 右对齐）
+：显示数值的正负符号%+d

awk操作符

算数操作符：

x+y,x-y,x*y,x/y,x^y,x%y（x%y，即x÷y取余数）

-x：转换为负数（-用双引号引起来，可以在0前面加上-号）

+x：转换为数值（同-，可以用双引号引起来）

字符串操作符：没有符号的操作符，字符串连接

赋值操作符：

=，+=，-=，*=，/=，%=，^=，++，--（++每次加1，–每次减1）

比较操作符：

>，>=，<，<=，!=，==

模式匹配符：

~：左边包含右边匹配到的内容
!~：左边不包含右边匹配到的内容

逻辑操作符

或 ||，与 &&，非!

PATTERN：根据pattern条件，过滤匹配的行，再做处理
如果未指定：空模式，匹配每一行
/正则表达式/：仅处理能够被模式匹配到的行，需要用 / / 括起来

关系表达式：结果有”真”有”假”；结果为”真”才会被处理；
真：结果为非0值，非空字符串
假：结果为空字符串
